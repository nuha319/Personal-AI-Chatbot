This is a smart chatbot designed using Python fundamentals (loops, conditionals, and dictionaries). A keyword-matching logic is implemented to make it behave like a real AI assistant and 
built modular functions for reusability. This is a rule-based chatbot, it doesn't use real AI models yet, but it behaves intelligently using Python logic.

Features:
1. Responds to simple user inputs
2. Works continuously until the user types bye
3. Easy to customize with new keywords and responses
4. Demonstrates real-world chatbot logic using Python fundamentals
